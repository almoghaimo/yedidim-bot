DEV: (SANDBOX)

the file "yedidim-eventCategories-sandbox.json" is a json representing the hirerchy of the events' categories and their sub-categories.

in order to update the categories:
1) make the desire changes to the json
2) go the the url: https://console.firebase.google.com/u/0/project/yedidim-sandbox-2/database/yedidim-sandbox-2/data/eventCategories
3) press on the 3 dots icon
4) choose "Import Json"
5) upload the new Json file.
